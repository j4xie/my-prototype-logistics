📋 库存管理
实时库存跟踪 — 批次追溯、预警、调拨
每个物料批次独立追踪：入库数量、已使用量、已预留量、剩余可用量。 系统自动预警：低库存预警（剩余 ≤ 30%）、临期预警（≤ 7 天到期）。 支持跨仓库/跨工厂调拨，状态全程可追踪。
示例：高筋面粉 MB-2026-0221-001，入库 200kg，已用 45kg，预留 30kg， 可用库存 125kg，存放于常温库-B区-货架1，有效期至 2026-05-21
4
⚙ 生产消耗
物料领用 — 生产批次自动关联原料消耗
车间开始生产时，根据 BOM（物料清单）自动关联所需原料批次。 领用后库存自动扣减，生产完成后成品自动入库到「成品库存」，生成新的成品批次。 关联报工数据，形成「原料 → 生产 → 成品」完整追溯链。
追溯链路： 原料批次 MB-001 → 生产批次 PB-2026-0221-001 → 成品批次 FG-001 → 报工记录 WR-001
