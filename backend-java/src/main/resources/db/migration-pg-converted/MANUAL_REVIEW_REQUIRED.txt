Common Manual Conversions:
-------------------------

1. INSERT IGNORE INTO -> INSERT INTO ... ON CONFLICT (column) DO NOTHING

2. REPLACE INTO -> INSERT INTO ... ON CONFLICT (column) DO UPDATE SET ...

3. GROUP_CONCAT(col) -> STRING_AGG(col, ',')

4. LIMIT offset, count -> LIMIT count OFFSET offset

5. ENUM('a','b','c') -> Create TYPE or use CHECK constraint:
   CREATE TYPE status_enum AS ENUM ('a', 'b', 'c');
   -- or --
   column VARCHAR(20) CHECK (column IN ('a', 'b', 'c'))

6. FULLTEXT INDEX -> PostgreSQL full-text search:
   CREATE INDEX idx_name ON table USING GIN(to_tsvector('english', column));

7. COMMENT='...' -> COMMENT ON COLUMN table.column IS '...';
