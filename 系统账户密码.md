## 🔧 常见问题

### 1. 登录失败？
- 检查用户名是否正确
- 确认密码是否为 `Admin@123456`（工厂用户）或 `123456`（平台管理员）
- 检查后端服务是否运行（`http://localhost:3001/health`）

### 2. 数据显示为 `--`？
- 部分统计数据需要后端API支持
- 平台管理员的工厂和用户数据已实现
- 其他统计功能正在开发中

### 3. Prisma Studio打不开？
- 需要手动启动: `cd backend && npm run studio`
- 访问: `http://localhost:5555`
- 确保MySQL服务正在运行
