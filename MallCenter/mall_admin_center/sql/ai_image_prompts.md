## API 调用示例

### DALL-E 3 (OpenAI)

```python
import openai

def generate_banner(prompt, theme_code):
    response = openai.Image.create(
        model="dall-e-3",
        prompt=prompt,
        size="1792x1024",  # Closest to 5:2 ratio
        quality="hd",
        n=1
    )
    return response.data[0].url
```

### Midjourney

```
/imagine prompt: [提示词内容] --ar 5:2 --v 6 --q 2
```

### Stable Diffusion

```python
from diffusers import StableDiffusionPipeline

def generate_image(prompt):
    pipe = StableDiffusionPipeline.from_pretrained("stabilityai/stable-diffusion-xl-base-1.0")
    image = pipe(
        prompt=prompt,
        width=1500,
        height=600,
        num_inference_steps=50
    ).images[0]
    return image
```

### DeepSeek (if available)

```python
import requests

def generate_with_deepseek(prompt):
    response = requests.post(
        "https://api.deepseek.com/v1/images/generations",
        headers={"Authorization": f"Bearer {API_KEY}"},
        json={
            "prompt": prompt,
            "size": "750x300",
            "n": 1
        }
    )
    return response.json()["data"][0]["url"]
```

## 后续步骤

1. **选择图片生成服务** - 根据预算和质量要求选择 DALL-E 3 / Midjourney / SD
2. **批量生成图片** - 使用上述提示词为每个主题生成 3-5 张 Banner 和 2 张背景
3. **上传到 OSS** - 将生成的图片上传到阿里云 OSS 对应目录
4. **更新数据库** - 将实际的图片 URL 更新到 `decoration_image_library` 表中
5. **测试验证** - 在商城前端验证图片加载和显示效果
