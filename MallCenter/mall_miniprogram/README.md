# 食品商城小程序

## 项目介绍

食品商城微信小程序，提供生鲜食品、加工食品、酒水饮料的在线购买服务。

## 功能模块

- 商品浏览与搜索
- 商品分类
- 购物车
- 订单管理
- 收货地址管理
- 优惠券
- 溯源查询
- AI智能推荐
- 商户中心
- 推荐系统

## 技术栈

- 微信小程序原生开发
- 后端: Spring Boot + MyBatis Plus
- 数据库: MySQL

## 开发说明

1. 配置 `config/env.js` 中的服务器地址
2. 使用微信开发者工具导入项目
3. 确保 AppID 配置正确
