如何导出数据报表？

您可以通过以下步骤导出数据报表：

  1. 进入「AI分析」-「数据报表」页面
  2. 选择需要的报表类型（生产/库存/质量/财务）
  3. 设置筛选条件（时间、产品、部门等）
  4. 点击右上角「导出」按钮
  5. 选择导出格式（Excel/PDF）
提示：导出的文件将保存在手机下载目录中
