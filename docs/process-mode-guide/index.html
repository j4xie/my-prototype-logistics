11
App端报工审批
主管在手机上直接审批工人的报工

进入审批页面

在工序任务列表页，点击顶部的 「审批图标」（剪贴板打勾图标）进入报工审批页面。

页面显示所有待审批的报工记录，包含报工人、数量、工序类别、提交时间。补报会标有黄色「补报」标签。

审批操作

  • 单条通过：点击「通过」按钮
  • 驳回：点击「驳回」→ 输入驳回原因 → 确定
  • 批量通过：勾选多条记录 → 顶部出现「批量通过」按钮
审批后记录从列表中消失，工人端的30秒轮询会自动检测到审批结果并弹出提示。
12
主管代报工
帮不会用手机的工人提交报工

什么时候需要代报工？

水产加工车间里年纪大的工人可能不会操作手机App。主管可以代替他们提交报工。

操作方式（Web管理端）

  1. 在Web管理端调用报工接口时，传入 targetWorkerId 参数指定实际工人
  2. 报工记录的 workerId 会记录为该工人，而非主管自己
  3. 后续审批流程不变
代报工功能通过API参数实现，后续会增加专用的代报工界面。

常见问题
看不到「工序任务」按钮？▼
需要管理员开启「工序生产模式」（系统管理 → 功能模块配置 → 生产管理 → mode 设为 PROCESS）。
任务列表是空的？▼
管理员还没生成当天的任务。请联系管理员在「产品-工序配置」页点「生成工序任务」。
报工提交后一直「待审批」？▼
所有报工都需要主管在Web端审批。请联系主管打开「生产管理 → 报工审批」。
报工被驳回了怎么办？▼
查看驳回原因（现在主管驳回时必须填写原因），修正后重新报工即可。
工人不会用手机怎么办？▼
主管可以使用「代报工」功能，通过传入 targetWorkerId 参数代替工人提交报工。报工记录会正确关联到该工人。
主管不在电脑前能审批吗？▼
可以！App新增了「报工审批」页面，在工序任务列表顶部的审批图标进入，支持单条通过、驳回和批量通过。
超过计划量了还能报吗？▼
可以。超过150%弹确认框，点「仍然提交」继续。
网络不好提交失败会重复吗？▼
不会。30秒内相同数量自动去重。
签到提示「该员工已签到」？▼
先签退再重新签到。
跨天数据会丢吗？▼
不会。完成量累加，有每小时自动校准。
怎么改工序顺序？▼
Web端「产品-工序配置」页用「上移」「下移」按钮。
工作流画布怎么连线？▼
从一个节点的输出端口拖拽到另一个节点的输入端口即可创建连线。点击连线可编辑转换事件、守卫条件和允许角色。
