# 工序生产操作手册

> 本手册面向工厂管理员和一线工人，从零开始一步步教你使用工序生产系统。
> 如果遇到问题，请参考末尾的「常见问题」。

---

## 第一部分：管理员操作（电脑端 Web-Admin）

管理员负责：创建工序 → 给产品关联工序 → 生成工序任务 → 审批报工

---

### 步骤 1：登录系统

1. 打开浏览器，输入管理系统地址
2. 输入**用户名**和**密码**
3. 点击 **「登 录」** 按钮

![登录页面](vue-01-login.png)

> 提示：如果是测试环境，可以点击下方「工厂总监」快捷登录按钮一键填入账号密码。

---

### 步骤 2：进入首页仪表板

登录成功后自动进入首页。这里可以看到今日概览数据。

![首页仪表板](vue-02-dashboard.png)

**你需要关注的区域：**
- 左侧菜单栏 → 找到「系统管理」
- 今日产量、完成批次、设备运行等数据卡片

---

### 步骤 3：创建工序（第一次使用才需要）

> 工序就是生产中的每个步骤，比如：拆箱分拣 → 卤制加工 → 真空包装

**操作路径：** 左侧菜单 → **系统管理** → **工序管理**

![工序管理页面](vue-03-work-processes.png)

**具体操作：**

1. 点击右上角 **「+ 新增工序」** 按钮
2. 填写以下信息：
   - **工序名称**（必填）：如「拆箱分拣」「卤制加工」「真空包装」
   - **工序类别**：从下拉菜单选择（前处理/加工/包装/灭菌/质检/存储/配送/其他）
   - **产出单位**（必填）：如「箱」「kg」「袋」「车」「框」
   - **预估工时**：该工序大概需要多少分钟
   - **排序**：数字越小越靠前
3. 点击 **「确定」** 保存

> 每道工序的**单位可以不同**！比如拆箱按「箱」，卤制按「kg」，包装按「袋」。

**管理已有工序：**
- **编辑**：修改工序名称、单位等信息
- **禁用/启用**：暂时不用的工序可以禁用，不会删除
- **删除**：彻底删除（不可恢复）

---

### 步骤 4：给产品关联工序

> 不同产品需要不同的工序流程。比如「带鱼段」需要：拆箱 → 卤制 → 包装

**操作路径：** 左侧菜单 → **系统管理** → **产品-工序配置**

![产品-工序配置（空状态）](vue-04-product-process-config.png)

**具体操作：**

1. 左侧**选择产品**：从下拉框选择要配置的产品（如「带鱼段」）
2. 右侧显示「暂无关联工序，请从下方添加」
3. 在「可添加的工序」区域，点击每道工序右边的 **「+ 添加」**
4. 按顺序依次添加：拆箱分拣 → 卤制加工 → 真空包装
5. 添加后可以用 **「上移」「下移」** 按钮调整顺序

![产品已关联3道工序](vue-05-product-process-linked.png)

> 上图显示「带鱼段」已关联3道工序，每道工序有编号（1/2/3）、类别标签和单位。

---

### 步骤 5：生成工序任务（每天排计划时用）

> 这一步是**每天都要做的**——把明天要生产的产品拆分成具体的工序任务。

**在步骤4的页面上操作：**

1. 确保已选择产品且已关联工序
2. 点击右上角绿色的 **「生成工序任务」** 按钮
3. 弹出确认框：「确定为「带鱼段」生成 3 道工序任务？」
4. 点击 **「确定」**
5. 看到提示 **「已生成 3 个工序任务」** 表示成功

> 生成后，工人在手机 App 上就能看到这些任务了。

---

### 步骤 6：审批报工

> 工人提交的报工需要管理员审批后才计入完成量。

**操作路径：** 左侧菜单 → **生产管理** → **报工审批**

![报工审批页面](vue-06-approval.png)

**审批操作：**

- **单条通过**：点击该行的 **「通过」** 按钮
- **驳回**：点击 **「驳回」** → 输入驳回原因 → 确定
- **批量通过**：勾选多条记录 → 点击顶部 **「批量通过」** 按钮

**审批后的效果：**
- 通过 → 该报工数量从「待审批」移到「已完成」
- 驳回 → 该报工数量从「待审批」中减去，工人可重新报工

## 第二部分：工人操作（手机端 App）

工人只需要做三件事：**签到** → **报工** → **查看进度**

---

### 步骤 7：打开 App 看首页

打开手机上的「白垩纪AI Agent」App，登录后看到首页。

![RN App 首页](rn-01-home.png)

**首页关键信息：**
- **待审批报工**：红色数字显示有多少报工等待主管审批
- **工序任务**：青色数字显示当前有多少个工序任务
- 底部 Tab 栏：首页 / AI分析 / 报表 / 智能分析 / **管理** / 我的

---

### 步骤 8：进入工序任务

**操作路径：** 底部点 **「管理」** Tab → 找到 **「工序任务」** 图标 → 点击进入

![管理页面](rn-02-management.png)

> 在「核心管理」区域可以看到「工序任务」的青色图标，点击即可进入。

---

### 步骤 9：查看工序任务列表

进入后看到所有工序任务的卡片列表。

![工序任务列表](rn-03-task-list.png)

**页面说明：**
- **顶部搜索框**：输入工序名搜索
- **筛选标签**：「进行中」「已完成」「全部」切换查看
- **每张卡片显示**：
  - 工序名称（如「卤制加工」）
  - 产品名和单位
  - **计划量** / **已完成** / **待审批**
  - 蓝色进度条
  - **「报工」按钮**（蓝色）

> 第一次进入会弹出操作指引：「1.选择工序卡片 2.点击报工 3.输入产出数量 4.提交后等待审批」

---

### 步骤 10：提交报工（最重要的操作！）

**这是工人每天最频繁的操作 -- 每小时报一次产量。**

1. 在任务列表找到要报工的工序卡片
2. 点击卡片右下角的 **「报工」** 按钮
3. 进入报工页面：
   - **产出数量**（必填）：输入这次的产出量
   - 可以点击**快捷填入按钮**（显示剩余量/半量）直接填入
   - 备注（选填）
4. 点击底部 **「提交报工」** 按钮
5. 看到提示 **「报工已提交，等待审批」** 表示成功
6. 点击 **「确定」** 返回任务列表

**特殊情况：**
- **超量报工**：如果输入的数量超过剩余量的150%，会弹出确认框，点击「仍然提交」可以继续
- **补报**：对已完成的任务也可以补报，会标记为「补报」需要审批
- **重复提交保护**：30秒内提交相同数量会被自动识别为重复

---

### 步骤 11：查看任务进度

**报工后想看进度？两种方式：**

**方式一：在列表直接看**
- 每张卡片的进度条和数字会实时更新

**方式二：点击卡片看详情**
- 点击任意卡片进入详情页
- 显示：计划量 / 已完成 / 待审批 三个大数字
- 下方显示所有报工记录和工人统计
- **每30秒自动刷新** -- 主管审批后会弹出提示「有报工已审批完成」

---

### 步骤 12：查看历史记录

点击任务列表右上角的 **时钟图标** 进入历史页面，可以查看所有已完成和已关闭的任务。

## 常见问题

### Q1：看不到「工序任务」按钮？
**A：** 需要工厂管理员在系统后台开启「工序生产模式」。请联系管理员操作。

### Q2：任务列表是空的？
**A：** 管理员还没有生成当天的工序任务。请联系管理员在 Web 端「产品-工序配置」页面点击「生成工序任务」。

### Q3：报工提交后一直「待审批」？
**A：** 所有报工都需要主管在 Web 端审批。请联系主管打开「生产管理 → 报工审批」页面进行审批。

### Q4：报工被驳回了怎么办？
**A：** 驳回原因会显示在记录中。修正后重新报工即可，驳回的数量不计入完成量。

### Q5：怎么修改工序顺序？
**A：** 在 Web 端「系统管理 → 产品-工序配置」页面，选择产品后，使用「上移」「下移」按钮调整工序顺序。

### Q6：超过计划量了还能报吗？
**A：** 可以。超过剩余量的150%时会弹出确认框，点击「仍然提交」即可。超量部分同样需要审批。

### Q7：网络不好提交失败了？
**A：** 系统有30秒防重复保护。请等待几秒后重试。如果重复提交了相同的数量，系统会自动识别并返回之前的记录，不会重复计数。

### Q8：签到时提示「该员工已签到」？
**A：** 说明该员工尚未签退。请先执行签退操作后再重新签到。

### Q9：一个产品要做好几天，跨天的数据会丢吗？
**A：** 不会。工序任务的完成量是累加的，跨天的每次报工都会正确累加。系统还有每小时自动校准机制确保数据准确。

### Q10：多个产品同时生产，任务会混在一起吗？
**A：** 任务列表会显示所有产品的任务。可以用搜索框输入产品名或工序名来过滤。
