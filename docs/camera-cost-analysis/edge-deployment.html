性价比冠军
方案 A: Jetson Orin Nano Super

  • • 处理 20 台摄像头 (ABE 优化后 1.4 req/s): 勉强够用 (~2-3 req/s 吞吐)
  • • 超低功耗 (7-25W)，无风扇设计
  • • 可直接安装在生产车间

方案 B: Jetson AGX Orin 64GB
¥14,400 / 台
  • • 运行 Qwen2.5-VL-7B，64GB 统一内存无显存瓶颈
  • • 可同时运行多个模型（手势 + OCR + 告警）
  • • 处理 20 台摄像头 (B+E 场景 3.8 req/s): 基本够用
  • • vLLM 支持在 Jetson 上运行，TensorRT 可进一步加速

方案 D: RTX 4090 工控机 (¥20,000-25,000/台)
跑 7B 模型吞吐最高 (~40 tok/s)，场景 2 (3.8 req/s) 完全没有压力。但体积大、功耗高、噪音大，不太适合车间环境。适合放在工厂机房，通过网络连接摄像头。

⚖ 优劣势分析

✓ 优势

  • 零网络依赖 - 断网照样跑
  • 数据不出厂 - 安全合规，适合食品工厂审计要求
  • 零延迟 - 本地处理最快
  • 长期最省 - 一次性投入后只付电费
  • 适合食品工厂的安全审计要求

✗ 劣势

  • 前期硬件投入
  • 每个工厂需要基本的设备运维
  • 模型更新需要逐厂推送
  • 设备故障需要现场处理或备件
